전투 컨텐츠 기획 정리

자동 전투 - 플레이어 : n대의 함선 조합
전투 스탯 - 체력, 공격력, 방어력, 명중, 회피, 치명타 확률, 치명타 피해, 속성 피해 보너스
그 외 스탯 - 함급, 슬롯(장비 장착 가능량), 성급, 레벨
함급 분류 : 초계함, 구축함, 순양함, 전함, 타이탄
성급 분류 : 4성, 5성

전투 공식
실 피해량 : 공격력 * 스킬 계수 * 속성 피해 보너스 * 치명타 계수 * 받는 피해량 * 랜덤 피해량 계수 +=15%
받는 피해량 = (1 / Mathf.Exp(-Mathf.Log(0.5f) / 1000 * 방어력)) * 적 명중 / (적 명중 + 회피)

전투 슬롯 갯수 : 초계함 4, 구축함 6, 순양함 8, 전함 10, 타이탄 12

함선 뽑기 시스템 : 보석을 사용해서 함선 가챠를 할 수 있음. 기본 드랍 확률 : 강화재료들 90%, 4성 함선 9%, 5성 함선 1%. 10연챠마다 4성 확정드랍, 80연챠마다 5성 확정드랍
함선 스킬 : 함선별 고유 스킬: 예) 4A1 : 장착한 모든 무기의 피해량을 20초동안 50% 증가시킨다. 쿨타임 30초
함선 강화 시스템 : 함선은 1부터 90레벨까지 강화할 수 있음. 함선 강화 재료가 소비되며 함선의 체력, 공격력, 방어력이 상승함.
함선 스킬 강화 시스템 : 함선의 고유 스킬은 1레벨부터 10레벨까지 강화할 수 있음. 스킬 강화 재료가 소비되며, 스킬의 효과가 증가함.

무기 피해 속성
물리, 광학, 입자, 플라즈마

스텟 적용 규칙 : 모든 고정 수치를 합산 후 % 보너스를 적용한다. 각 % 보너스는 모두 합산된다.

장비 분류 (주스텟 랜덤 1종)
추진기 주스텟 : 회피%, 속성 피해 보너스&, 공격력%, 추진기 슬롯에 장착 가능
반응로 주스탯 : 체력%, 방어력%, 치명타 확률%, 치명타 피해%, 공격력%, 반응로 슬롯에 장착 가능
라디에이터 주스탯 : 체력%, 방어력%, 공격력%, 라디에이터 슬롯에 장착 가능
무기 주스텟 : 공격력, 전투 슬롯에 장착 가능
장갑 주스탯 : 방어력, 전투 슬롯에 장착 가능

장비 강화 시스템 : +20강까지 가능. 강화 단계에 따라 주스탯 강화. 4강마다 옵티마이즈 스탯 언락. 옵티마이즈 최대 5회 가능
옵티마이즈 스탯 종류 : 공격력%, 공격력, 방어력%, 방어력, 체력%, 체력, 치명타 확률, 치명타 피해, 명중%, 회피%
장비 추출 : 필요없는 장비를 추출해서 장비 강화 재료로 환원.

무기 스킬 : 해당 무기의 평타. 무기마다 다름. 예) 고정형 레일건 Mk1 : 전방 20도의 각도를 향해 포탄을 발사하여 공격력 120%의 물리 피해를 입힙니다. 쿨타임 5초, 사거리 500km
무기 조정 : 같은 종류의 무기를 소모해서 무기 스킬의 데미지를 올림. 조정은 최대 4회 진행하여 조정도를 +5까지 확대 가능. 조정시마다 25%의 데미지가 증가되며, 최대 100% 증가됨.

전투 알고리즘: 장비의 세트 효과. 2세트 및 6세트 효과 존재.
알고리즘 알파 2세트 : 물리 피해 20% 증가. 6세트 : 500km 안쪽에 적이 없을 경우, 물리 피해 60% 증가
알고리즘 베타 2세트 : 광학 피해 20% 증가. 6세트 : 모든 무기의 쿨타임 30% 감소
알고리즘 감마 2세트 : 입자 피해 20% 증가. 6세트 : 피해를 가한 적의 방어력을 10초동안 60%만큼 감소
알고리즘 델타 2세트 : 플라즈마 피해 20% 증가. 6세트 : 무기 발사 시마다 플라즈마 피해량 15초동안 10% 증가. 최대 6중첩 가능
알고리즘 엡실론 2세트 : 방어력 40% 증가, 6세트 : 방어 태세 시 모든 무기의 쿨타임 60% 감소
알고리즘 제타 2세트 : 공격력 20% 증가, 6세트 : 치명타 확률 20% 증가, 치명타 피해 40% 증가

전투 태세 : 전투 중, 전 함대 혹은 각 함선별 전투 태세 지정 가능.
집중 사격 태세 : 모든 무장을 주 대상을 집중 공격하는 데에 집중. 주 목표가 사거리 밖이라면 무장을 사용하지 않음
공격 태세 : 함선을 우선적으로 공격. 사거리 내에 함선이 없을 경우 투사체를 요격.
수호 태세 : 투사체를 우선적으로 요격. 사거리 내에 투사체가 없을 경우 함선을 공격
방어 태세 : 가장 가까운 사거리 내 투사체를 요격. 함선을 대상으로 공격하지 않음.

아웃게임 컨텐츠 정리

재화 종류
에너지 크레딧 : 인게임 재화.
슈퍼 크레딧 : 고급 인게임 재화 : 가챠에 사용됨.
메타 물질 : 함선에 경험치를 추가해서 함선을 강화할 수 있음. 함선 스킬을 강화하는데에 사용.
연구 샘플 : 장비 강화, 장비 옵티마이즈에 사용됨
연료 : 레진. 최대 240개, 6분당 1씩 회복됨. 하루에 240씩 회복됨.

추적 섬멸 임무 : 던전 같은거. 연료를 40 소모해서 입장할 수 있음.
추적 섬멸 - 정예 : 함선 강화 재료 파밍 가능
추적 섬멸 - 의뢰 : 에너지 크레딧 파밍 가능
추적 섬멸 - [세트이름] 분석 : [세트이름] 세트효과를 가진 랜덤한 장비를 파밍 가능
탐색 임무 : 자동 파밍. 함선이 쭉 전진하며 소행성을 격파해서 에너지 크레딧을 파밍할 수 있음. 가끔씩 함선 재료와 장비 재료가 드랍됨.
