전투 컨텐츠 기획 정리

자동 전투 - 플레이어 : n대의 함선 조합
전투 스탯 - 체력, 공격력, 방어력, 치명타 확률, 치명타 피해, 속성 피해 보너스
그 외 스탯 - 함급, 슬롯(장비 장착 가능량), 성급, 레벨
함급 분류 : 초계함, 구축함, 순양함, 전함, 타이탄
성급 분류 : 4성, 5성

전투 공식
실 피해량 : 공격력 * 스킬 계수 * 속성 피해 보너스 * 치명타 계수 * 받는 피해량 * 랜덤 피해량 계수 +=15%
받는 피해량 = (1 / Mathf.Exp(-Mathf.Log(0.5f) / 1000 * 방어력))

함선 뽑기 시스템 : 보석을 사용해서 함선 가챠를 할 수 있음. 기본 드랍 확률 : 강화재료들 90%, 4성 함선 9%, 5성 함선 1%. 10연챠마다 4성 확정드랍, 80연챠마다 5성 확정드랍
함선 스킬 : 함선별 고유 스킬: 예) 4A1 : 장착한 모든 무기의 피해량을 20초동안 50% 증가시킨다. 쿨타임 30초
함선 강화 시스템 : 함선은 1부터 90레벨까지 강화할 수 있음. 함선 강화 재료가 소비되며 함선의 체력, 공격력, 방어력이 상승함.
함선 스킬 강화 시스템 : 함선의 고유 스킬은 1레벨부터 10레벨까지 강화할 수 있음. 스킬 강화 재료가 소비되며, 스킬의 효과가 증가함.

무기 피해 속성
물리, 광학, 입자, 플라즈마

스텟 적용 규칙 : 모든 고정 수치를 합산 후 % 보너스를 적용한다. 각 % 보너스는 모두 합산된다.

장비 분류 (주스텟 랜덤 1종)
추진기 주스텟 : 체력%, 방어력%, 속성 피해 보너스&, 공격력%, 추진기 슬롯에 장착 가능
반응로 주스탯 : 체력%, 방어력%, 치명타 확률%, 치명타 피해%, 공격력%, 반응로 슬롯에 장착 가능
라디에이터 주스탯 : 체력%, 방어력%, 공격력%, 라디에이터 슬롯에 장착 가능
무기 주스텟 : 공격력, 전투 슬롯에 장착 가능
장갑 주스탯 : 방어력, 전투 슬롯에 장착 가능

장비 강화 시스템 : +20강까지 가능. 강화 단계에 따라 주스탯 강화. 4강마다 옵티마이즈 스탯 언락. 옵티마이즈 최대 5회 가능
옵티마이즈 스탯 종류 : 공격력%, 공격력, 방어력%, 방어력, 체력%, 체력, 치명타 확률, 치명타 피해
장비 추출 : 필요없는 장비를 추출해서 장비 강화 재료로 환원.

무기 스킬 : 해당 무기의 평타. 무기마다 다름.

전투 장비 장착 : 무기 및 방어구는 각 함선별로 최대 5개씩 장착할 수 있음.
장비는 코스트가 존재함. 함선의 전투 장비는 모든 장비의 코스트 총합이 함선의 수용량을 초과할 수 없음.

전투 알고리즘: 장비의 세트 효과. 2세트 및 3세트 효과 존재.
알고리즘 알파 1세트 : 공격력 +18%. 2세트 : 물리 피해 보너스 +18%, 4세트 : 자신의 체력이 90% 이상일 경우, 물리 피해 보너스 36%만큼 증가
알고리즘 베타 1세트 : 방어력 +24%. 2세트 : 광학 피해 보너스 +18%, 4세트 : 모든 무기의 쿨타임 30%만큼 감소
알고리즘 감마 1세트 : 치명타 피해 +24%. 2세트 : 입자 피해 보너스 +18%, 4세트 : 입자 피해를 가한 적의 방어력을 12초동안 60%만큼 감소, 중첩되지 않음
알고리즘 델타 1세트 : 치명타 확률 +12%. 2세트 : 플라즈마 피해 보너스 +18%, 4세트 : 무기 발사 시마다 플라즈마 피해 12초동안 8%만큼 증가. 최대 6중첩 가능

아웃게임 컨텐츠 정리

재화 종류
에너지 크레딧 : 인게임 재화.
슈퍼 크레딧 : 고급 인게임 재화 : 함선 가챠에 사용됨.
연료 : 레진. 최대 240개, 6분당 1씩 회복됨. 하루에 240씩 회복됨.

임무 : 던전 같은거. 연료를 40 소모해서 입장할 수 있음.
협약 의뢰 : 대량의 크레딧 파밍 가능
추적 섬멸 - 알파 분석 : 알파 세트효과를 가진 랜덤한 장비 혹은 실탄류 무기 파밍 가능
추적 섬멸 - 베타 분석 : 베타 세트효과를 가진 랜덤한 장비 혹은 레이저류 무기 파밍 가능
추적 섬멸 - 감마 분석 : 감마 세트효과를 가진 랜덤한 장비 혹은 입자류 무기 파밍 가능
추적 섬멸 - 델타 분석 : 델타 세트효과를 가진 랜덤한 장비 혹은 플라즈마류 무기 파밍 가능

탐색 임무 : 자동 파밍. 함선이 쭉 전진하며 적 함선을 격파해서 에너지 크레딧을 파밍할 수 있음
일정 시간마다 보스가 등장하며, 보스를 격파할 시 탐색 레벨이 증가하며 나오는 적 함선의 레벨 및 보상이 증가함.
보스 격파에 실패하면 보스 등장이 보류되며 보스 등장 버튼이 활성화됨. 해당 버튼을 눌러 보스를 재소환 가능.
